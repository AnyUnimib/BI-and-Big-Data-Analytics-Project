Airbnb – Redis Project (Big Data & Business Intelligence)

How to run the project:

1. Upload all files from this folder to Google Colab.
2. Open the notebook: Standalone_Script.ipynb
3. Click: Runtime → Run all

What happens automatically:
- Required libraries are installed
- Redis server is installed and started
- Airbnb data is downloaded and processed
- Redis database is populated
- Queries and visualizations are executed
- Results are generated automatically

Where to see the results:
- All outputs are generated in the main notebook:
  BIG_DATA_&_BI.ipynb during execution.
- After execution, an executed notebook file named:
  executed_BIG_DATA_&_BI.ipynb
  is also created in the Colab Files panel.
- This executed notebook contains the same results and can be opened or downloaded to review the full output.

No manual setup or configuration is required.
